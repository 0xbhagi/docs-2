# 代币标准

## 接口

每当我们对被触发的事件日志的子集感兴趣的时候(无论是哪个原始合约)，Dune都会使用interface-decoding去处理。典型的的例子包括`ERC20`、`ERC721`和`ERC1155`的转账事件。 此方法被保留下来用于特殊情况。 这些表可以轻松跟踪流入和流出合约以及钱包的同质化以及非同质化代币并在Dune中被广泛使用。
