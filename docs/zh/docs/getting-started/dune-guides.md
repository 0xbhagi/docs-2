## 用 Dune 调取数据的一般流程

1. **整理一份所有相关合约的清单，并提交进行**[**解码**](../features/decoded-contracts.md)**。**
2. **使用 SQL 查询调取数据**
3. **将结果可视化**
4. **在看板上组装可视化内容**
5. **修饰看板**
6. **享受数据**
