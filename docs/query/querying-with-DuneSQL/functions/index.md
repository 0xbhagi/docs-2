title: Functions and Operators
description: DuneSQL supports a wide range of built-in functions and operators.
---

In this section, you will learn about the functions and operators supported by DuneSQL.

Since DuneSQL is a fork of Trino, it supports all of the functions and operators supported by Trino. In addition, DuneSQL supports a few functions and operators that are not supported by Trino.
